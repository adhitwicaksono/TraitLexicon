# TraitLexicon Controlled Vocabulary

This folder stores controlled vocabulary files used by TraitLexicon.

These YAML files define recommended values for:

- kingdom scopes
- trait categories
- evidence levels
- curation statuses
- candidate module confidence labels
- relationship types
- gene roles
- context fields
- trait synonyms
- placeholder-gene warnings

The goal is to keep TraitLexicon consistent before resolver code is added.
