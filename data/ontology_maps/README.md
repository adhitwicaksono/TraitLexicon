# TraitLexicon Ontology Maps

This folder stores seed mappings between TraitLexicon modules and external ontology or annotation vocabularies.

Current seed files include:

- `go_terms_seed.yaml`
- `kegg_terms_seed.yaml`
- `plant_ontology_seed.yaml`
- `plant_trait_ontology_seed.yaml`
- `mapman_terms_seed.yaml`
- `pfam_interpro_seed.yaml`

These files are **not final ontology mappings**.

They are starter mapping files and should be verified before publication-level use.
